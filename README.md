Данный проект представляет из себя кастомный плагин для AutoDesk Revit. Суть плагина - расчет зон дополнительного армирования на основании нагрузки в узлах, выделенных при применении метода конечных элементов.
Функционал плагина позволяет задавать арматурные данные (шаг, цену, длину), сохранять их между пользовательскими сессиями, проводить расчеты зон для выбранных уровней модели в плитах перекрытия.
Основа - генетический алгоритм.
Стек - C#, WPF, RevitAPI.
